HTML is also known as HyperText MarkUp Language. It is popularly also known as the web language and mostly used for web development.
CSS is called as the Cascading Style Sheet. Its is the language used popularly for styling the HTML web page created to produce awesome effects and visuals.
JavaScript as known as JS is the DOM language used for addressing elements and tags of the HTML web page thus, providing an intrinsic and a dynamic outlook.
The Career options after learning these languages and their frameworks are the jobs of the Back end, Front end and the Full Stack developer.
UX design means the design related to the User Associativity. The art of UX designing renders the web page or application more dynamic and enhances productivity.
UI design means the designing related to the User Interface. This enables and improves the User interactivity with the web page or the web application in numerous ways.
Front end developer is the developer of projects that works at the front-end and knows the architectural frameworks of Java, HTML, Js, Node, etc thereby, having a keen knowledge of the front working of the application
A Full Stack Web developer is the developer of projects that work at both front end and back end and knows many intrinsic frameworks and architectures. Thereby it hones the art of entire application manipulation.
